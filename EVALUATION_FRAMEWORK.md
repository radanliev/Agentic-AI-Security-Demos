# Evaluation Framework

## Metrics
- Attack Success Rate (ASR)
- Persistence Across Turns
- Detection Rate
- Mitigation Latency

## Methodology
1. Define baseline agent behaviour
2. Introduce adversarial input
3. Measure deviation and persistence
4. Apply mitigation
5. Re-evaluate performance

## Benchmark Goals
- Reproducibility
- Comparability across systems
- Standardisation of agentic AI security testing
