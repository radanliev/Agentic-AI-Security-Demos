# Week 3 Slides to Demo Map

This map aligns the week 3 slides in the bootcamp deck to the new demos in this repository.

| Slide range | Slide theme | Demo file |
|---|---|---|
| 77–78 | Week 3 learning outcomes and standards framing | `README.md` / `controls_matrix.md` |
| 79–80 | Agentic AI architecture and capability boundaries | `demos/capability_boundary_demo.py` |
| 81–83 | Trust boundaries, A2A security, and threat modelling | `demos/trust_boundary_demo.py` and `demos/a2a_impersonation_demo.py` |
| 84–90 | Architectural vulnerabilities and multi-stage attack chains | `demos/attack_chain_demo.py` |
| 91–95 | Secure autonomous infrastructure and runtime monitoring | `demos/runtime_protection_demo.py` |

## Why these demos fit the slides

- The slides now centre on agent reasoning, tools, memory, A2A communication, and runtime control.
- The demos therefore avoid classic MLOps-only examples and instead show how autonomous agent systems fail and how they are defended.
- The repo is deterministic and toy-scale so it can be run safely in class.
