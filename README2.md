# Agentic AI Security Demos

A curated collection of **reproducible security demonstrations for agentic AI systems**, covering adversarial attacks, system-level vulnerabilities, and defensive design patterns across autonomous and multi-agent architectures.

## Scope
This repository focuses on:
- Prompt injection and jailbreak attacks
- Data poisoning in retrieval and memory layers
- Tool misuse and orchestration-layer exploits
- Agent-to-agent (A2A) attack surfaces
- Defensive controls and secure design patterns

## Structure
- `DEMO_INDEX.md` — catalogue of all demonstrations
- `THREAT_MODEL.md` — formal threat modelling for agentic systems
- `EVALUATION_FRAMEWORK.md` — benchmarking and metrics
- `QUICKSTART_EXPERIMENTS.md` — minimal reproducible demos

## Positioning
This repository serves as a **reference testbed for agentic AI security evaluation**, bridging:
- academic research
- red teaming practices
- operational AI security engineering

See `QUICKSTART_EXPERIMENTS.md` for minimal working examples.
