# Week 4 Demo Files

This folder contains home-based exercises for the Week 4 homework.

## Files

### `prompt_injection_chain.py`
Demonstrates how an indirect instruction can propagate across multiple agents and reach execution.

### `policy_engine_demo.py`
Shows runtime enforcement of A2A governance rules, including explicit deny policies.

### `unsafe_action_logger.py`
Logs agent activity, flags anomalous actions, and writes structured event records.

## How to run

Use Python 3.10+.

```bash
python demo/prompt_injection_chain.py
python demo/policy_engine_demo.py
python demo/unsafe_action_logger.py
