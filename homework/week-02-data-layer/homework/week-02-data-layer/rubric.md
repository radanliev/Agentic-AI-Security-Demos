Grading rubric (recommended)

Each task scored out of 10; total 40.

- Correctness & technical accuracy (4)
- Demonstrable artefacts / runnable code (3)
- Threat realism & attack modelling (2)
- Clarity and actionable mitigations (1)

Bonus (up to 5 points)
- Extra credit for telemetry pipeline that raises an alert automatically upon detection
- Extra credit for a short unit test suite validating detection logic
